## corne keymap
this repo contains source for my corne v3

features include:

- [x] colemak-dh layout
- [x] 42 key layout, utilizing 6 extra keys
- [x] all non essential keys moved to tri layer
- [x] qwerty game layer
- [ ] timeless home-row mods
